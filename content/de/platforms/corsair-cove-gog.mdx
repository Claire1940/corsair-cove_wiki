Um das Spiel mit GOG-Dateien einzurichten, folgen Sie diesen Schritten, um eine saubere Installation zu gewährleisten.

<Steps>
  <Step num="1" title="Installationsdateien herunterladen">
    Loggen Sie sich über einen Webbrowser oder den GOG Galaxy-Client in Ihr GOG-Konto ein. Navigieren Sie zu Ihrer Bibliothek, suchen Sie das Spiel und laden Sie die Offline-Standalone-Installationsdateien herunter. Speichern Sie alle Teile im selben Verzeichnis auf Ihrem lokalen Laufwerk.
  </Step>
  <Step num="2" title="Setup-Datei ausführen">
    Doppelklicken Sie auf die ausführbare Hauptinstallationsdatei. Wählen Sie Ihr bevorzugtes Installationsverzeichnis. Es wird empfohlen, das Spiel auf einem Solid State Drive (SSD) zu installieren, um die Ladezeiten beim Wechsel zwischen der Siedlungskarte und der Weltkarte zu verkürzen.
  </Step>
  <Step num="3" title="Optionale Patches anwenden">
    Prüfen Sie die GOG-Bibliotheksseite auf Hotfixes oder Day-One-Patches. Falls verfügbar, laden Sie diese Installer herunter und führen Sie sie aus, nachdem das Hauptspiel installiert wurde, um die Kompatibilität mit dem neuesten Build sicherzustellen.
  </Step>
  <Step num="4" title="Start-Einstellungen konfigurieren">
    Navigieren Sie zum Installationsordner und klicken Sie mit der rechten Maustaste auf die ausführbare Spieldatei. Wählen Sie „Eigenschaften“ und aktivieren Sie das Kontrollkästchen, um das Programm als Administrator auszuführen. Dies verhindert potenzielle Schreibberechtigungsprobleme beim Speichern Ihres Spielstands.
  </Step>
</Steps>

<Callout type="warning" title="Sicherheit der Speicherstände">
Standalone-Installer synchronisieren Spielstände nicht automatisch mit der Cloud, es sei denn, Sie starten das Spiel über den GOG Galaxy-Client. Sichern Sie regelmäßig Ihr Speicherverzeichnis in Ihrem lokalen AppData-Ordner, um Datenverlust zu vermeiden.

<Section icon={Sparkles} title="Performance-Optimierung & Fehlerbehebung">

Obwohl das Spiel für eine breite Palette von Hardware optimiert ist, können große Siedlungen mit hunderten von Crewmitgliedern Performance-Einbußen erleiden. Das Anpassen spezifischer Grafik- und Simulations-Einstellungen kann helfen, eine stabile Bildrate beizubehalten.

| Einstellung | Empfohlener Wert | Performance-Auswirkung | Details |
| --- | --- | --- | --- |
| Schattenqualität | Mittel | Hoch | Das Verringern reduziert den CPU-Overhead erheblich. |
| Detailgrad der Menge | Niedrig/Mittel | Hoch | Reduziert die Rendering-Komplexität der Crewmitglieder. |
| V-Sync | Ein | Niedrig | Verhindert Screen-Tearing auf Standardmonitoren. |
| Simulationsgeschwindigkeit | Normal | Variabel | Vorspulen erhöht die CPU-Last bei Pfadfindungsberechnungen. |

Falls Probleme beim Starten des Spiels über den GOG-Installer auftreten, konsultieren Sie die folgenden Schritte zur Fehlerbehebung.

| Problem | Mögliche Ursache | Lösung |
| --- | --- | --- |
| Spiel stürzt beim Start ab | Fehlende DirectX-Runtime-Dateien | Führen Sie den DirectX-Web-Installer von Microsoft aus. |
| Spielstände laden nicht | Berechtigungseinschränkungen | Führen Sie die ausführbare Spieldatei als Administrator aus. |
| Schwarzer Bildschirm beim Start | Nicht unterstützte Auflösung | Bearbeiten Sie die Konfigurationsdatei in AppData passend zu Ihrem Monitor. |
| Ruckeln während des Gameplays | Konflikte mit Hintergrundanwendungen | Deaktivieren Sie Overlays von Drittanbietern und schließen Sie unnötige Apps. |
