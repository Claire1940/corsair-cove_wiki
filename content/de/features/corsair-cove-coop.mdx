<Section icon={Swords} title="Seekampf und Erkundung">

  <Card title="Kampfwerte">

    - **Rüstung**: Reduziert den erlittenen Gesamtschaden.

<Section icon={Shield} title="Piraten-Zusammenhalt aufrechterhalten">

Der Zusammenhalt (Cohesion) ist das Lebenselixier deines Betriebs. Er repräsentiert die Einheit deiner Besatzung. Wenn diese Leiste auf Null sinkt, ist das Spiel vorbei.

| Ereignistyp | Auswirkung auf Zusammenhalt | Gegenmaßnahme |
| :--- | :--- | :--- |
| **Wenig Nahrung/Alkohol** | Ständiger Abzug | Mehr Kombüsen/Tavernen bauen |
| **Interne Fehden** | Plötzlicher Abfall | Münzen ausgeben, um Streitigkeiten beizulegen |
| **Schiffsangriffe** | Massiver Abfall | Verteidigungskanonentürme unterhalten |
| **Erfolgreiche Raubzüge** | Moderater Zuwachs | Erkundungsziele abschließen |
