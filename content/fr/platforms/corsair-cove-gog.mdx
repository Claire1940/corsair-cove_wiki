<Section icon={Shield} title="Configuration requise et installation de Corsair Cove GOG">

Avant de lancer votre empire pirate, assurez-vous que votre système répond aux exigences matérielles. Comme le jeu intègre des systèmes de simulation complexes pour les chaînes de production et les comportements individuels de l'équipage, les performances du processeur (CPU) sont critiques en fin de partie, lorsque votre colonie s'agrandit.

| Configuration requise | Configuration minimale | Configuration recommandée |
| --- | --- | --- |
| Système d'exploitation | Windows 10 (64 bits) | Windows 10/11 (64 bits) |
| Processeur | Intel Core i3-6100 / AMD FX-8350 | Intel Core i5-8400 / AMD Ryzen 5 2600 |
| Mémoire | 8 Go de RAM | 16 Go de RAM |
| Carte graphique | GTX 760 / Radeon R9 280X | GTX 1060 / RX 580 |
| DirectX | Version 11 | Version 12 |
| Stockage | 4 Go d'espace disponible | 4 Go d'espace SSD disponible |

Pour installer le jeu à l'aide des fichiers GOG, suivez ces étapes pour garantir une installation propre.

<Steps>
  <Step num="1" title="Télécharger les fichiers d'installation">
    Connectez-vous à votre compte GOG via un navigateur Web ou le client GOG Galaxy. Accédez à votre bibliothèque, localisez le jeu et téléchargez les fichiers d'installation hors ligne autonomes. Enregistrez toutes les parties dans le même répertoire sur votre disque local.
  </Step>
  <Step num="2" title="Exécuter le fichier d'installation">
    Double-cliquez sur le fichier exécutable d'installation principal. Choisissez votre répertoire d'installation préféré. Il est recommandé d'installer le jeu sur un disque SSD pour réduire les temps de chargement lors des transitions entre la carte de la colonie et la carte du monde.
  </Step>
  <Step num="3" title="Appliquer les correctifs optionnels">
    Consultez la page de la bibliothèque GOG pour d'éventuels correctifs ou patchs Day One. Si disponibles, téléchargez et exécutez ces installateurs une fois l'installation du jeu principal terminée pour garantir la compatibilité avec la dernière version.
  </Step>
  <Step num="4" title="Configurer les paramètres de lancement">
    Accédez au dossier d'installation et faites un clic droit sur l'exécutable du jeu. Sélectionnez les propriétés et cochez la case pour exécuter le programme en tant qu'administrateur. Cela évite d'éventuels problèmes d'autorisation d'écriture lors de la sauvegarde de votre partie.
  </Step>
</Steps>

<Callout type="warning" title="Sécurité des fichiers de sauvegarde">
Les installateurs autonomes ne synchronisent pas automatiquement les sauvegardes sur le cloud, sauf si vous lancez le jeu via le client GOG Galaxy. Sauvegardez régulièrement votre répertoire de sauvegarde situé dans votre dossier AppData local pour éviter toute perte de données.
</Callout>

</Section>

<Section icon={Sparkles} title="Optimisation des performances et dépannage">

Bien que le jeu soit optimisé pour une large gamme de configurations matérielles, les grandes colonies comptant des centaines de membres d'équipage peuvent subir des baisses de performances. L'ajustement de paramètres graphiques et de simulation spécifiques peut aider à maintenir un framerate stable.

| Paramètre | Valeur recommandée | Impact sur les performances | Détails |
| --- | --- | --- | --- |
| Qualité des ombres | Moyenne | Élevé | Réduire ce paramètre diminue considérablement la charge du processeur. |
| Détails de la foule | Bas/Moyen | Élevé | Réduit la complexité du rendu des membres de l'équipage. |
| V-Sync (Synchro verticale) | Activé | Faible | Évite les déchirements d'écran sur les moniteurs standards. |
| Vitesse de simulation | Normale | Variable | L'avance rapide augmente l'utilisation du processeur lors des calculs de recherche de chemin. |

Si vous rencontrez des problèmes pour lancer le jeu depuis l'installateur GOG, consultez les étapes de dépannage ci-dessous.

| Problème | Cause potentielle | Résolution |
| --- | --- | --- |
| Le jeu plante au démarrage | Fichiers d'exécution DirectX manquants | Exécutez l'installateur Web DirectX de Microsoft. |
| Les sauvegardes ne se chargent pas | Restrictions d'autorisation | Lancez l'exécutable du jeu en tant qu'administrateur. |
| Écran noir au lancement | Paramètres de résolution non pris en charge | Modifiez le fichier de configuration dans AppData pour correspondre à votre moniteur. |
| Saccades pendant le jeu | Conflits d'applications en arrière-plan | Désactivez les superpositions tierces et fermez les applications en arrière-plan inutiles. |
