<Section icon={Target} title="生産チェーンの最適化">

コーヴ（入り江）で生き残るためには、絶え間ない資源の流れが必要です。増え続ける海賊の人口のニーズと、艦隊からの要求のバランスを取らなければなりません。

| 資源レベル | 例 | 生産元 |
| :--- | :--- | :--- |
| **基本** | 木材、石材、食料 | 木こり、焚き火 |
| **加工** | 板、石レンガ | 製材所、石切り場 |
| **高度** | 黒色火薬、帆 | 缶詰工場、帆職人 |
| **贅沢品** | ラム酒、タバコ | 蒸留所、農場 |

<Steps>
  <Step num="1" title="基本ニーズの確保">
    結束力の低下を防ぐため、すぐに焚き火を作り、食料を集めましょう。
  </Step>
  <Step num="2" title="物流の確立">
    道路や崖の小道を使って、資源建物を海賊のキャンプに接続します。
  </Step>
  <Step num="3" title="生産のスケールアップ">
    安定したら、船の建造に備えて製材所と倉庫を建設します。
  </Step>
  <Step num="4" title="新しい島への拡張">
    船を使って近くの島を襲撃し、自国で生産できない資源を獲得します。
  </Step>
</Steps>

<Callout type="warning" title="効率アラート">
常に資源の生産量と消費量を監視してください。倉庫の容量が満杯になったり、供給ラインが切断されたりすると、建物は稼働を停止し、不満の連鎖を引き起こします。
</Callout>

</Section>

<Section icon={Swords} title="海戦と探索">

コルセア・コーヴの海戦はターン制で、物資と乗組員の体力を管理する必要があります。七つの海でのあらゆる決断が、あなたの海賊としてのレガシーに影響を与えます。

<CardGrid cols={2}>
  <Card title="戦闘ステータス">
    - **砲撃ダメージ**: 敵の耐久力にダメージを与えます。
    - **接舷ダメージ**: 敵の乗組員を直接攻撃します。
    - **速度/機動性**: 敵の砲撃を回避するために使用します。
    - **装甲**: 受けるダメージ全体を軽減します。
  </Card>
  <Card title="戦術物資">
    - 物資は「アクションポイント」として機能します。
    - 物資が尽きると、戦闘の選択肢が制限されます。
    - 物資を補給するには、多くの場合、体力や乗組員を犠牲にする必要があります。
  </Card>
</CardGrid>

<Callout type="success" title="戦略">
交戦する前に必ず敵船のステータスを確認してください。自船の火力が劣っている場合は、接舷攻撃に集中して敵の乗組員を無力化し、自船のダメージを最小限に抑えましょう。
</Callout>

</Section>

<Section icon={Shield} title="海賊の結束力の維持">

結束力（Cohesion）は運営の命綱です。これは乗組員の団結力を表しています。このゲージがゼロになると、ゲームオーバーになります。

| イベントタイプ | 結束力への影響 | 対策戦略 |
| :--- | :--- | :--- |
| **食料/酒の不足** | 持続的な低下 | ギャレー（調理場）や酒場をさらに建設する |
| **内部の確執** | 急激な低下 | コインを支払って紛争を解決する |
| **船の襲撃** | 大幅な低下 | 防衛用の砲台を維持する |
| **襲撃の成功** | 緩やかな上昇 | 探索目標を完了する |
