- **主なキーワード**: **コルセア・コーヴの祭壇 (corsair cove altar)** の検索は、多くの場合、深い瞑想状態にあるベダビンの遊牧民NPC**「アルタルキズ (Altarkiz)」**を指しています。
- **アクセス要件**: トック船長（Captain Tock）を通じてコーヴへアクセスするには、クエスト**「海賊の呪い (The Corsair Curse)」**を開始する必要があります。
- **資源エリア**: イチイ（Yew）とカエデ（Maple）の木をアンロックするには、**「ドラゴン・スレイヤーI (Dragon Slayer I)」**の完了とハリス（Haris）への会話が必要です。
- **戦闘のハイライト**: ダンジョンにはレベル82の**オーガレス・ウォーリア（Ogress Warriors）**が出現し、F2P（無課金）における優秀な金策対象となっています。
- **祈りの利便性**: 町の中に物理的な祈りの祭壇（Prayer altar）は存在しませんが、**コルセア・コーヴの銀行**が資源管理の主要ハブとなっています。

<Section icon={Map} title="コルセア・コーヴの探索とアルタルキズの場所">

**コルセア・コーヴの祭壇**エリアのレイアウトを理解することは、無課金（F2P）と課金（Members）プレイヤーの両方にとって重要です。フェルディップ・ヒルズ（Feldip Hills）の南岸に位置するこの孤立した港は、重要な訓練場として機能しています。多くのプレイヤーが伝統的な祈りの祭壇を探しますが、最も注目すべき「祭壇（altar）」との関連は、深い集中状態にあり、通常の会話ができない遊牧民のNPC**「アルタルキズ (Altarkiz)」**です。

| 特徴 | 要件 | 場所 |
| :--- | :--- | :--- |
| **初期アクセス** | 「海賊の呪い」の開始 | ポート・サリム / リミングトン |
| **NPCアルタルキズ** | なし | 町の南部 |
| **銀行へのアクセス** | 「海賊の呪い」の完了 | コーヴ中央 |
| **資源エリア** | 「ドラゴン・スレイヤーI」 | 港の北西部 |

<Callout type="tip" title="移動のヒント">
クエストを開始していない場合は、ポート・サリムの北にいるトック船長（Captain Tock）に話しかけてください。開始後、彼はアクセスしやすいようにリミングトンの西端に移動します。
</Callout>

<Section icon={Swords} title="コルセア・コーヴのダンジョンとオーガレス戦闘">

中心街のすぐ西には、**コルセア・コーヴのダンジョン**の入り口があります。ここは中レベルのF2P戦闘訓練における最高の目的地です。ダンジョンにはオーガレス・ウォーリアとオーガレス・シャーマン（Ogress Shamans）が生息しており、高ティアの防具やルーンを含む価値あるドロップテーブルで知られています。

**オーガレスのドロップテーブルのハイライト:**
| アイテム | レア度 | 価値 / 用途 |
| :--- | :--- | :--- |
| **ルーンのフルヘルム** | レア | ハイアルケミ / 戦闘 |
| **リンプワートの根** | コモン | ハーブロア / 利益 |
| **シャーマンの仮面** | レア | 装飾用 / コレクション |
| **法則/自然のルーン** | アンコモン | 実用 / 利益 |

<Steps>
  <Step num="1" title="準備">
    ダンジョン内の岩の影から安全に攻撃できるため、食べ物とレンジ（遠隔）またはマジック（魔法）装備を持参してください。
  </Step>
  <Step num="2" title="入場">
    中心街の西にある穴を見つけてください。「呪い」の会話を避けるために、必要なクエストの手順を完了していることを確認してください。
  </Step>
  <Step num="3" title="位置取り">
    石筍（せきじゅん）を利用して、オーガレス・ウォーリアの近接攻撃を阻みます。シャーマンは魔法を使用するため、高い魔法防御が推奨されます。
  </Step>
</Steps>

<Callout type="warning" title="危険エリア">
オーガレス・シャーマンは魔法で正確に攻撃してきます。レベルが低い場合は、出口の近くにとどまるか、可能であれば「魔法防御（Protect from Magic）」の祈りを使用してください。
</Callout>

</Section>

<Section icon={Anchor} title="移動方法とセーリングのユーティリティ">

2026年のメタにおいて、コルセア・コーヴは**セーリング（Sailing）**スキルの重要なハブとなっています。ドックでは、造船所サービスやバウンティタスクなど、船乗りのための様々な機能が提供されています。最初の「海賊の呪い」をクリアしていれば、セーリングレベル40のプレイヤーがここにドッキングできます。

| 移動手段 | 要件 | 目的地 |
| :--- | :--- | :--- |
| **チャーター船** | クエストの完了 | 世界各地の港 |
| **アジリティの岩** | アジリティ10 | フェルディップ・ヒルズ |
| **資源の岩** | アジリティ30 | 資源エリア |
| **手漕ぎボート** | セーリング51 / 建設50 | アングラーズ・リトリート |

<Callout type="success" title="運賃割引">
クエスト**「キャビン・フィーバー (Cabin Fever)」**を完了し、**「シャロスの指輪(a) (Ring of Charos(a))」**を装備することで、チャーター船の運賃がそれぞれ50%ずつ累積で割引されます。
</Callout>

</Section>
