<Section icon={Map} title="序盤のキャンプ管理をマスターする">

コルセア・コーヴで海賊の植民地を成功させるには、急速な拡大と乗組員の士気の微妙なバランスが必要です。帝国の基盤は海賊たちの満足度にかかっています。満足度が低下すると、キャンプは混乱に陥り、生産が停止し、最悪の場合は反乱につながる可能性があります。これを管理する最も効果的な方法は、正確な建物の配置とタイミングです。

**動画のハイライト:**
- 最初のキャンプ建設中は、スペースキーを使用して時間を停止させる。
- 供給施設（密造酒蒸留所/キャンプファイア）をサービス施設（酒場/調理場）のすぐ隣に建設する。
- 最初の満足度を100%にするために、少なくとも6つのテントをメインハブに接続する。
- 生産地点と消費地点の間の最短ルートを監視する。

新しいキャンプを開始するとき、「一時停止戦略」は最大の味方です。ゲームを一時停止することで、満足度メーターが中立のうちに、資源や時間を消費することなくインフラ全体をレイアウトできます。これにより、「混乱」状態が発生するのを未然に防ぐことができます。

| 建物タイプ | 目的 | 配置のヒント |
| :--- | :--- | :--- |
| **海賊キャンプ** | 中央ハブ | 資源クラスターの中心 |
| **密造酒蒸留所** | 酒場への供給 | 酒場の直上または直下 |
| **キャンプファイア** | 調理場への供給 | 調理場の直上または直下 |
| **テント** | 居住区 | 道路で接続、最低6つ |
| **調理場** | 食事サービス | 迅速な輸送のためキャンプファイアの近く |
| **酒場** | 飲料サービス | 迅速な輸送のため蒸留所の近く |

<Callout type="tip" title="効率の秘密">
密造酒蒸留所などの供給施設を酒場のすぐ上または下に配置すると、労働者の移動経路が最短になります。これにより、サービス施設の稼働時間が最大化され、満足度を常に100%に保つことができます。
</Callout>

<Section icon={Swords} title="造船と乗組員の維持">

ゲームが進み、**デストロイヤー（Destroyer）、ハンター（Hunter）、プラウラー（Prowler）、リーヴァー（Reaver）**などの高度な船のクラスがアンロックされると、物流の要件は飛躍的に増大します。基本的な船とは異なり、高度な艦船はステータスに大きなボーナスを与える専門の乗組員を利用しますが、彼らは特定の維持資源を要求します。これらのニーズを無視すると、長期キャンペーンで最も強力な船が役に立たなくなってしまいます。

<CardGrid cols={2}>
  <Card title="スカリーワグ＆クロウズアイ">
    - **要件**: カッサバパンと専用の衣装。
    - **メリット**: 高い接舷戦闘力と向上した視認範囲。
    - **維持**: 医薬品とブーツの安定した供給が必要。
  </Card>
  <Card title="ピカルーン＆ボンバルディア">
    - **要件**: 高品質なラム酒と火薬。
    - **メリット**: 絶大な砲撃ダメージと船体保護。
    - **維持**: 床屋と外科医への継続的なアクセスが必要。
  </Card>
</CardGrid>

これらのスペシャリストの雇用は、アンロックした建物と連動しています。例えば、デストロイヤー級の船を建造すると、サトウキビ農園と樽職人（Cooper）がアンロックされます。これらがなければ、エリート乗組員を満足させるために必要なラム酒を生産することはできません。

| 船のクラス | 主要な乗組員 | 必要な維持アイテム | 提供されるボーナス |
| :--- | :--- | :--- | :--- |
| **デストロイヤー** | ボンバルディア | ラム酒、火薬 | +15% 砲撃ダメージ |
| **プラウラー** | クロウズアイ | パン、ブーツ | +20% 機動力 |
| **リーヴァー** | スカリーワグ | 医薬品、ラム酒 | +10% 接舷戦闘力 |
| **スワビー** | デフォルト乗組員 | なし | 基本性能 |

<Callout type="warning" title="ロジスティクス・アラート">
艦隊をアップグレードする前に、乗組員のホバーメニューで「雇用コスト」と「維持費」を確認してください。ラム酒や医薬品の生産ラインが整っていない場合、新しい船は進水直後から士気低下のペナルティを受けることになります。
</Callout>

<Section icon={Target} title="生産チェーンと専門化">

初心者がよく犯す間違いの一つは、すべての海賊キャンプを自給自足にしようとすることです。コルセア・コーヴでは、経済を繁栄させる鍵は専門化にあります。キャンプが基本的なサービス施設を通じて100%の満足度を維持している限り、単一のタイプの資源の抽出と加工に完全に集中することができます。

<Steps>
  <Step num="1" title="地域資源の評価">
    マップビューをズームアウトし、「すべて」を押して利用可能な抽出ポイントを確認します。グアノ、サトウキビ、または木材が豊富な島を特定します。
  </Step>
  <Step num="2" title="インフラの確立">
    労働力が安定するように、標準的な「満足度コア」（酒場、調理場、蒸留所、キャンプファイア）を建設します。
  </Step>
  <Step num="3" title="生産の専門化">
    残りのスペースを特定のチェーンに充てます。例えば、「火薬の島」には炭焼き窯とグアノ抽出所を設置します。
  </Step>
  <Step num="4" title="ネットワークの接続">
    艦隊を使用して、生産ハブから乗組員の雇用に必要なキャンプへ、専門品（ラム酒やパンなど）を輸送します。
  </Step>
</Steps>

| 資源チェーン | 必要な建物 | 最終製品 | 用途 |
| :--- | :--- | :--- | :--- |
| **アルコール** | サトウキビ農園、樽職人、蒸留所 | ラム酒 | 酒場の供給、乗組員の維持 |
| **兵装** | グアノ採取場、炭焼き窯、火薬工場 | 火薬 | 戦闘、船のアップグレード |
| **繊維** | 麻農園、ロープ製造所、仕立屋 | ロープ、ブーツ | 造船、乗組員の衣装 |
| **医療** | 薬草園、研究所、外科医 | 医薬品 | 高度な乗組員の維持 |

<Callout type="success" title="最適化のヒント">
カッサバパンだけを生産しているキャンプも、火薬を生産しているキャンプと同じくらい価値があります。すべての島を多様化させなければならないというプレッシャーを感じる必要はありません。物流が正しく処理されていれば、専用の「パンの島」が帝国全体を養うことができます。
</Callout>

<Section icon={Compass} title="探索とマップイベント">

コルセア・コーヴの世界には、植民地の海岸を越えた先に多くのチャンスが溢れています。キャンペーンマップを航海していると、課題や報酬を表すさまざまなアイコンに遭遇します。これらのマーカーを理解することは、生産だけに頼りすぎずに資源の備蓄を維持するために不可欠です。

**主要な探索マーカー:**
- **番号付きの島 (1, 2, 3)**: 戦闘やナラティブイベントの難易度を示します。
- **ダイヤモンドアイコン**: 公海上の「重要地点（Points of Interest）」を表します。
- **戦霧（Fog of War）**: 近くを航行して新しい島を明らかにする必要があるエリアです。

| イベントタイプ | アイコン | 潜在的な報酬 | リスクレベル |
| :--- | :--- | :--- | :--- |
| **漂流貨物** | ダイヤモンド | 無料のラム酒、木材、またはロープ | ゼロ |
| **難破船** | ダイヤモンド | 無料の海賊、ゴールド、材料 | 低 |
| **島イベント** | 番号付き | 大量の資源キャッシュ、新しい技術 | 変動 |
| **敵のパトロール** | 船アイコン | 戦闘経験値、戦利品 | 高 |

<Callout type="info" title="エキスパートのアドバイス">
常にプラウラーのような高速船を「偵察」専用に用意しておきましょう。それを使って、できるだけ多くのダイヤモンドアイコンを回収してください。これらの無料資源は、生産不足の際のギャップを埋めたり、新しいキャンプに必要な追加の海賊を提供したりすることがよくあります。
</Callout>

<Section icon={Shield} title="戦闘戦略と船のステータス">

ゲームでは特定のキャンペーンイベントに対して「推奨される船」が提示されることが多いですが、これらの推奨は絶対ではありません。海戦の成功は、船のレベルやクラス名よりも、実際の属性（砲撃ダメージ、接舷戦闘力、速度、機動力、装甲）に大きく依存します。

**主要な戦闘属性:**
- **砲撃ダメージ（Gun Damage）**: 遠距離から敵をどれだけ早く沈められるかを決定します。
- **接舷戦闘力（Boarding Strength）**: 船を拿捕し、最大の戦利品を回収するために不可欠です。
- **機動力（Maneuverability）**: 舷側砲を向けるためにどれだけ早く旋回できるかに影響します。
- **装甲（Armor）**: 飛来する砲火に対する主要な防御力です。

| 船のステータス | 重要度 | 最適な用途 |
| :--- | :--- | :--- |
| **装甲** | 高 | 近接接舷中の攻撃に耐える |
| **速度** | 中 | 逃走する商船を追い詰める |
| **機動力** | 高 | 激しい砲火を避け、位置取りを確保する |
| **接舷** | 変動 | 戦闘からの資源獲得を最大化する |

<Callout type="tip" title="ステータス比較">
ミッションで装甲10のレベル2リーヴァーが推奨されていても、手持ちのレベル1デストロイヤーが装甲16でより高い砲撃ダメージを持っているなら、デストロイヤーを選んでください。現在のメタでは、生のステータスは常に「推奨」ラベルに勝ります。
</Callout>

</Section>
