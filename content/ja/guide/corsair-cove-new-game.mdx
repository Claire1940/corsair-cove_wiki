<Section icon={Map} title="コルセア・コーヴ 新作ゲーム：概要とゲームプレイの基本">

Hooded Horseがパブリッシャーを務める、待望の新作ゲーム**『コルセア・コーヴ（Corsair Cove）』**は、サバイバル都市建設ジャンルに新鮮なアプローチをもたらします。海賊の黄金時代の終焉を舞台に、プレイヤーは容赦ない海賊ハンター「アマラ」が率いるスペインの囚人船から脱出し、荒涼とした無人島に漂着したキャストウェイ（漂流者）となります。生き残るためには、必死の漂流者集団から強力な海賊シンジケートへと成長しなければなりません。

平坦で広大な土地に依存する従来の都市建設ゲームとは異なり、本作では「垂直方向」に思考を巡らせる必要があります。ロープ、梯子、橋の複雑なネットワークを駆使して、切り立った崖の側面に不安定ながらもバランスを保った荒屋（シャンテ）、生産施設、防衛システムを建設していくことになります。

**動画の見どころ:**
- **垂直建設**: 崖沿いの梯子や橋を利用して、海岸沿いのキャンプと高地にある資源を接続する方法を学びます。
- **生産チェーン**: 最初の木こり、製材所、倉庫ネットワークを構築するためのステップバイステップのガイド。
- **乗組員の満足度**: キャンプファイア、調理場（ギャレー）、酒場がどのように連携して、乗組員の結束（コヒージョン）メーターを満たすかを確認します。
- **初期の戦闘**: ターン制の船対船カードバトルシステムと、ティーチのようなキャプテンの配置方法を紹介。

<Section icon={Target} title="複雑な生産チェーンとレイアウトの管理">

物流は海賊の隠れ家の命綱です。新作ゲーム『コルセア・コーヴ』では、資源は必要な場所に自動的にテレポートしません。代わりに、「運び手（フェッチャー）」と呼ばれる専用の労働者クラスが、採取場所から加工施設、そして最終的には流通拠点へと物理的にアイテムを運ぶ必要があります。

平地が不足しているため、崖沿いに通路を建設する必要があります。建物が梯子や橋を介して中央の物流グリッドに接続されていない場合、労働者はアイドル状態になり、経済が停滞してしまいます。

<CardGrid cols={3}>
  <Card title="垂直方向の拡張">
    - **垂直の梯子を活用**して、低地のビーチと高地の森林を繋ぎましょう。
    - **崖沿いの通路を建設**して、豊富な鉄や銅の鉱床にアクセスします。
    - 高地の作業拠点の近くに住宅を配置し、**移動時間を最小限に抑えます**。
  </Card>
  <Card title="グリッドの接続性">
    - 建設メニューで**接続線を確認**し、緑色のルートがアクティブであることを確認します。
    - 裂け目に直接ロープブリッジを架けることで、**遠回りを避けます**。
    - 運び手が急な地形で立ち往生した場合は、**ルートの問題を監視**します。
  </Card>
  <Card title="備蓄バッファ">
    - 製材所などの高出力施設の近くに**倉庫を配置**します。
    - 資材置き場に**特定の枠を指定**し、乱雑になるのを防ぎます。
    - 早期に**古い難破船を撤去**し、海岸近くの平坦な建設スペースを確保します。
  </Card>
</CardGrid>

工業セクターを設計する際は、安全性を念頭に置いてください。製材所や製粉所は大量の微粉塵を発生させ、これが重大な火災の危険因子となります。これらの建物が適切な間隔を空けずに居住区の近くに密集していると、一度の事故でコロニー全体が壊滅的な被害を受ける可能性があります。

| 製品 | 原材料 | 主な建物 | 主な消費者 |
| :--- | :--- | :--- | :--- |
| **板材（Planks）** | 原木 | 製材所 | 建設および造船 |
| **シチュー（Stew）** | 生の食材 | 調理場（ギャレー） | 漂流者および運び手 |
| **密造酒（Moonshine）** | 酵母、砂糖、水 | 密造酒蒸留所 | 酒場の客 |
| **黒色火薬（Black Powder）** | グアノ、硫黄 | 火薬調合所 | 砲塔および大砲 |

<Callout type="warning" title="工業災害の警告">
密造酒蒸留所を製材所のすぐ隣に配置しないでください。非常に可燃性の高いおがくずとアルコール蒸留の組み合わせは、インフラを破壊する壊滅的な工業火災を引き起こす原因となります。
</Callout>

</Section>

<Section icon={Trophy} title="乗組員の満足度、結束（コヒージョン）、および成長">

海賊の忠誠心は胃袋の満たされ具合に比例します。ゲームには、社会全体の士気を表す重要な「結束（コヒージョン）」メーターが存在します。結束がゼロになると、乗組員は反乱を起こし、激しい決闘や建物の破壊工作を招き、最悪の場合はキャンペーンが早期に終了してしまいます。

高い結束を維持するためには、人口の進化する要求を満たす必要があります。基本的な「漂流者」はシンプルなテントと温かいシチューで満足しますが、高度な「専門家」や「エリート・コルセア」は、タバコ、上質な衣服、ポケットサイズの銃器などの贅沢品を求めます。

<Steps>
  <Step num="1" title="基本的なシェルターの確保">
    中央の海賊キャンプの近くにテントを建設します。これにより、新しく到着した漂流者に住処を提供し、ゲーム初期の結束の低下を防ぎます。
  </Step>
  <Step num="2" title="食料とエールの確保">
    調理場（ギャレー）を建設して生の食材をボリューム満点のシチューに加工し、密造酒蒸留所を設置して地元の酒場に強いアルコールを供給します。
  </Step>
  <Step num="3" title="桟橋の建設">
    海岸沿いに桟橋を建設し、最初の偵察船の組み立てを開始します。これにより、艦隊を派遣して近くの島々を襲撃し、現地で生産できない資源を略奪することができます。
  </Step>
  <Step num="4" title="専門家の育成">
    住宅をアップグレードして、大型軍艦の指揮に不可欠な航海士や砲撃手などの専門的な乗組員を誘致しましょう。
  </Step>
</Steps>

| 乗組員の階級 | 主な消費物 | 役割 | 基礎結束力への影響 |
| :--- | :--- | :--- | :--- |
| **漂流者（Drifters）** | 基本的なシェルター | 未割り当ての労働力 | **中立** |
| **運び手（Fetchers）** | シチュー、基本的なエール | 資源の輸送 | **低** |
| **専門家（Specialists）** | 医薬品、タバコ | 航海士、砲撃手 | **中程度** |
| **エリート・コルセア** | 宝飾品、上質な衣服 | 高度な海戦 | **高** |

<Callout type="success" title="結束力向上のコツ">
地元の酒場で定期的にフェスティバルを開催することは、海賊ハンターの襲撃を生き延びた後に、失われた結束力を急速に回復させる非常に効果的な方法です。
</Callout>

</Section>

<Section icon={Swords} title="海戦と防衛要塞">

海賊の隠れ家としての悪名が高まるにつれ、スペイン王室はあなたを壊滅させるために、ますます強力な艦隊を送り込んできます。これらの襲撃を生き延びるためには、攻撃的な海軍遠征と、防御的な島の要塞化の両方をマスターしなければなりません。

『コルセア・コーヴ』の新作ゲームにおける海戦は、戦術的なターン制のダイスとカードのシステムで解決されます。出航する前に、船を率いる有能なキャプテンを任命する必要があります。例えば、キャプテン・ティーチは攻撃的なプレイスタイルに最適です。彼のユニークな能力は、敵にダメージを与えるたびに戦闘で追加のダイスロールを可能にします。

敵の艦隊が到着した際にあなたの艦隊が遠征中であった場合、海岸沿いの砲塔が最後の防衛線となります。これらの砲塔は、大砲の弾と黒色火薬を砲塔に直接届ける機能的なサプライチェーンが確立されている限り、侵入してくる船を自動的に攻撃します。

| 建造物 | 資材コスト | 必要な弾薬 | 戦術的価値 |
| :--- | :--- | :--- | :--- |
| **砲塔（Gun Tower）** | 木の板、石ブロック | 黒色火薬 | スペイン艦隊から港を守る |
| **倉庫（Warehouse）** | 板材、鉄の補強材 | なし | 最大4種類の貨物を保管 |
| **桟橋（Pier）** | 重い木材、ロープ | なし | 艦船の建設と修理を行う |

<Callout type="info" title="戦術情報">
運び手が黒色火薬を時間内に届けられなかった場合、砲塔は完全に機能停止します。防衛ラインの近くには、常に弾薬専用の倉庫備蓄を確保しておきましょう。
</Callout>
