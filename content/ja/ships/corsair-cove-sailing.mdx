**動画ハイライト:**

- **探索メカニクス**: 強力な船を建造し、七つの海を探索して危険な報酬に立ち向かう。

<Section icon={Map} title="航路とアクセス要件">

コルセア・コーヴへ到達するには、クエストの進行とスキルのマイルストーンを達成する必要があります。無料プレイ（F2P）プレイヤーも入り江には入れますが、最も効率的な**コルセア・コーヴ航海**ルートは、地元のクエストラインで実力を証明したプレイヤーのみが利用可能です。フェルディップ丘陵周辺の航行は危険を伴うことがありますが、本土とこの孤立した港を結ぶいくつかの移動手段が存在します。

| 方法 | 要件 | 行き先 |
| :--- | :--- | :--- |
| **トック船長** | ポート・サリムの北 | 入り江への初回輸送 |
| **トック船長** | リミントン（西） | クエスト開始後の輸送 |
| **敏捷性ショートカット** | 敏捷性10 | フェルディップ丘陵の岩場 |
| **資源ショートカット** | 敏捷性30 | 資源エリアへのアクセス |
| **航海ドッキング** | 航海40 | コルセア・コーヴ主要埠頭 |
| **チャーター船** | クエスト完了 | ギルリナー各地の港 |

チャーター船システムはここでも非常に強力です。クエスト「Cabin Fever」を完了するか、Ring of Charos(a)を使用することで、航海コストを大幅に削減できます。これにより、中レベルプレイヤーにとって非常に費用対効果の高い拠点となります。

<Callout type="warning" title="アクセス制限">
敏捷性が30のプレイヤーは、資源エリア付近の岩場を利用する前に「Dragon Slayer I」を完了し、ハリスに話しかける必要があります。これらの二次要件を満たしていない場合、道は閉ざされたままとなります。
</Callout>

</Callout>
