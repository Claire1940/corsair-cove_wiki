<Section icon={Settings} title="2026年向け最適グラフィック設定">

『コルセア・コーヴ』をSteam Deckで最高に楽しむには、芸術的な海賊の美学と携帯ゲーム機の技術的制限とのバランスを取る必要があります。以下の設定は、GPUの熱を抑えつつ、「七つの海」を鮮明かつ詳細に保つために最適化されています。

| 設定項目 | 推奨値 | 影響度 |
| :--- | :--- | :--- |
| **解像度** | 1280x800（ネイティブ） | 高 - テキストを鮮明に保つ |
| **テクスチャ品質** | 中/高 | 中 - VRAMに依存 |
| **影の品質** | 低 | 高 - 大幅なFPS向上 |
| **水面の反射** | 中 | 中 - 海の雰囲気に不可欠 |

| **群衆の密度** | 中 | 高 - CPU負荷に致命的 |

<Section icon={Gamepad2} title="操作とナビゲーションの最適化">

『コルセア・コーヴ』はマウスとキーボード操作を前提に設計されているため、携帯ゲーム機で海賊の隠れ家をナビゲートするには、スマートなコントローラー設定が必要です。Steam Deckのトラックパッドはここで最大の武器となり、建物の正確な配置や乗組員の幸福度の管理を可能にします。

**推奨されるコントローラーマッピング：**

| Steam Deckのボタン | ゲーム内アクション | メリット |
| :--- | :--- | :--- |
| **右トラックパッド** | マウス移動 | 正確な建物の配置 |
| **左トラックパッド** | ラジアルメニュー（ホットキー） | 建設/マップ/乗組員へのクイックアクセス |
| **L4 / R4 グリップ** | ズームイン / アウト | 生産ラインの見やすさ向上 |
| **L5 / R5 グリップ** | ユニットの切り替え | 乗組員の素早い管理 |
| **Rトリガー** | マウスの左クリック | 主要なインタラクション |

<Steps>
  <Step num="1" title="トラックパッドマウスの有効化">
    「コントローラー設定」>「レイアウトを編集」に進みます。右トラックパッドを「マウス」に設定し、感度を中程度にして拠点のナビゲーションをスムーズにします。
  </Step>
  <Step num="2" title="ラジアルメニューの設定">
    左トラックパッドを仮想メニューに割り当てます。このメニューに数字の1〜5をマッピングし、建築カテゴリと探索マップを素早く切り替えられるようにします。
  </Step>
  <Step num="3" title="デッドゾーンの調整">
    強力な船で海を航海する際の「ドリフト」を防ぐため、ジョイスティックのデッドゾーンを10%に設定します。
  </Step>
</Steps>

『コルセア・コーヴ』をSteam Deckでプレイしながら海賊帝国を築くには、資源の生産と乗組員の士気に常に注意を払う必要があります。画面が小さいため、資源不足や海賊の不満に関する通知を見落としがちです。

| 資源タイプ | 管理のコツ | 携帯モードでの優先度 |
| :--- | :--- | :--- |

**生産ラインの効率化：**
- **グループ化**：乗組員の移動距離を最小限に抑えるため、ラム酒醸造所を砂糖プランテーションの近くに配置します。
- **倉庫**：ボトルネックを防ぐため、3〜4個の生産建物ごとに専用の倉庫を近くに配置します。
- **アップグレード**：専門的な建物の小さなアップグレード要件テキストを読むには、Steam Deckの「Steamボタン + L1」拡大鏡を使用します。
