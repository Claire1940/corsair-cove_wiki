<Section icon={Cpu} title="コルセア・コーヴ システム要件の概要">

険しい崖や熱帯の島々に広がる海賊の楽園を築くには、お使いのPCが公式の**コルセア・コーヴ システム要件**を満たしている必要があります。Limbic Entertainmentが開発し、Hooded Horseが販売するこの垂直都市建設ゲームは、詳細な海洋物理学、ダイナミックな天候、そして複雑な物流ネットワークを備えています。ジップライン、橋、個々の乗組員など、数千もの動くパーツをレンダリングするため、安定したフレームレートを維持するには十分な性能のプロセッサとグラフィックカードが不可欠です。

以下の表に、ゲームを実行するために必要なハードウェア構成をまとめています。基本的なプレイを目指す場合も、高画質でのパフォーマンスを求める場合も、2026年7月31日の公式リリース前にお手持ちのハードウェア構成とこれらの仕様を比較してください。

### 最小システム仕様

これらのコンポーネントは、低い解像度と控えめなグラフィック設定でゲームを実行するために必要です。この構成はゲーム序盤の入植地には適していますが、海賊の拠点が拡大するにつれてパフォーマンスが低下する可能性があります。

| コンポーネント | 最小仕様 |
| :--- | :--- |
| **OS** | Windows 10 (64ビット) |
| **CPU** | Intel Core i7-7700 / AMD Ryzen 5 1600 |
| **RAM** | 8 GB RAM |
| **GPU** | NVIDIA GeForce GTX 1660 Super (6 GB) / AMD Radeon RX 5600 XT (6 GB) |
| **DirectX** | バージョン 12 |
| **ストレージ** | 30 GB の空き容量 |

### 推奨システム仕様

1080p解像度かつ高いグラフィックプリセットでスムーズに動作させるため、開発者は以下のセットアップを推奨しています。このハードウェアであれば、ゲーム終盤の複雑な生産チェーンや大規模な艦隊戦も円滑に動作します。

| コンポーネント | 推奨仕様 |
| :--- | :--- |
| **OS** | Windows 10 (64ビット) |
| **CPU** | Intel Core i5-9600K / AMD Ryzen 5 3600 |
| **RAM** | 16 GB RAM |
| **GPU** | NVIDIA GeForce RTX 2070 (8 GB) / AMD Radeon RX 6600 XT (8 GB) |
| **DirectX** | バージョン 12 |
| **ストレージ** | 30 GB の空き容量 (SSDを強く推奨) |

<Callout type="warning" title="DirectX 12の要件">
コルセア・コーヴは、垂直レンダリングパイプラインを管理するためにDirectX 12 APIの機能に大きく依存しています。DX12をサポートしていない古いグラフィックカードでは、ゲームが起動しないか、深刻なグラフィックの乱れが発生します。
</Callout>

<Section icon={Sparkles} title="垂直都市建設とハードウェアのパフォーマンス">

コルセア・コーヴは、従来の平坦な都市建設ゲームではありません。プレイヤーは崖を登るように建物を建て、巨大な橋で島々を繋ぎ、エレベーターやジップラインを使って資源を垂直方向に運びます。この垂直性は、CPUとGPUの両方に影響を与える独特の処理上の課題をもたらします。

人口が増えるにつれ、何百人ものドリフター、ビルダー、フェッチャーが物流ネットワークを通じて物資を運びます。CPUは、複数の垂直階層にわたるリアルタイムの経路探索ルートを計算しなければなりません。物流ラインが長すぎたり非効率だったりして赤色に変わると、CPUの負荷が大幅に増加します。

大規模な拡張中もシステムを低温に保つために、パフォーマンスに影響を与える特定のグラフィック設定の調整を検討してください。

| 設定項目 | 推奨値 | パフォーマンスへの影響 |
| :--- | :--- | :--- |
| **シャドウ品質** | 中 | 密集した崖沿いの地区でのGPU負荷を軽減 |
| **群衆密度** | 中 | 海賊の活動がピークの時のCPU使用率を低下 |
| **水のディテール** | 高 | GPUコストを抑えつつ海洋の美観を維持 |
| **パス可視化** | オフ | CPUの経路探索負荷を軽減するためデフォルトで無効 |

<Callout type="tip" title="物流の最適化">
生産ルートを短く「緑色」に保つことは、経済を改善するだけでなく、経路探索の計算に費やされるCPUサイクルを減らすことにも繋がります。局所的な海賊キャンプや配送ハブを活用して、ハードウェアをスムーズに動作させましょう。
</Callout>

</Section>
