<Section icon={Sparkles} title="Performance Optimization & Troubleshooting">

While the game is optimized for a wide range of hardware, large settlements with hundreds of crew members can experience performance degradation. Adjusting specific graphical and simulation settings can help maintain a stable frame rate.

| Setting | Recommended Value | Performance Impact | Details |
| --- | --- | --- | --- |
| Shadow Quality | Medium | High | Lowering this reduces CPU overhead significantly. |
| Crowd Detail | Low/Medium | High | Reduces the rendering complexity of crew members. |
| V-Sync | On | Low | Prevents screen tearing on standard monitors. |
| Simulation Speed | Normal | Variable | Fast-forwarding increases CPU usage during pathfinding calculations. |

If you encounter issues launching the game from the GOG installer, consult the troubleshooting steps below.

| Issue | Potential Cause | Resolution |
| --- | --- | --- |
| Game crashes on startup | Missing DirectX runtime files | Run the DirectX web installer from Microsoft. |
| Saves not loading | Permission restrictions | Run the game executable as an administrator. |
| Black screen on launch | Unsupported resolution settings | Edit the configuration file in AppData to match your monitor. |
| Stuttering during gameplay | Background application conflicts | Disable third-party overlays and close unnecessary background apps. |
